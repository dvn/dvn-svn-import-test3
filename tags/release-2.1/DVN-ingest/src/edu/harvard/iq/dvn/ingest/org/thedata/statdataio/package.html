The main package of the StatData I/O API.

Many common StatData I/O operations may be performed using the static methods of the entry-point class StatDataIO.

This package contains a basic class for storing the contents of statistical data files (SDIOData); for controlling the reading process (StatDataFileReader) and writing process (StatDataFileWriter);

As of the release of DVN 2.0, the Dataverse Network Project implements its reading plug-in for STATA-DTA, SPSS POR and SPSSS SAV formats.

The wirter(output)-side functionality has not yet fully implemented and skeletal stub code only.
